If you choose to use the {{e.parentEntity}}'s uniqueKey, you will be limited to 1 {{e.conceptAlias}} for every {{e.parentEntity}}. {{w}} What worksheet do you want users to enter data on?. You can choose an existing worksheet, or create a new one. The advantage of using multiple worksheets is that you reduce duplicate data entry for "parent" entities (You only need to enter a row 1 time and the uniqueKey can be referenced in other worksheets). The disadvantage is that it requires more work on the users part by needing to associate identifier keys between sheets. Unique Across Project?
