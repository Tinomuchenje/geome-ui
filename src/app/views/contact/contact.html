Creating New Projects

If you would like information on creating a new proejct in GeOMe, send an email request to geome.help@gmail.com with a brief statement of what your project involves. The GeOMe team will be in touch about getting you setup.

Technical Support

If you have general questions about GeOMe, please read the User Help Document. There is also information about using GeOMe on the About page. If you still have questions of a technical nature, or encounter a bug, feel free to send an email to geome.help@gmail.com with your questions, along with as much information as possible so we can help resolve the issue.
