Genomic Observatories MetaDatabase (GEOME)

The Genomic Observatories Meta-Database (GEOME) is a web-based database which captures metadata on biological samples and associated genetic sequences. GEOME satisfies the requirements of the genomic observatories model, including capturing the who, what, where, and when associated with all samples. In addition to validating sample-based metadata, GEOME enables loading FASTA files describing Sanger sequences, as well as photos, and 96-well plate-based tissue matching. When working with metadata for FASTQ files from massively parallel sequencing runs, GEOME produces a submission package to ease the upload process into NCBIâ€™s Sequence Read Archive (SRA). GEOME then harvests the SRA accession numbers, creating a permanent link between the data and metadata.
